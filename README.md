## 💡 About This Project
A responsive, mobile-first portfolio built with **HTML5 and CSS3**, designed to present my skills, projects, and professional journey with clarity and interactivity.

**Key Features:**
- 🌗 Dark/Light Mode Toggle  
- 📊 Animated Skill Indicators  
- 🖥️ Project Showcases & Case Studies  
- 🏅 Verified Certification Badges (Credly)  
- 📱 Mobile-First Responsive Design  
- 📄 Downloadable CV
